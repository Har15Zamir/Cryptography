## Setup
```cmd
git clone https://github.com/Har15Zamir/cryptography.git
cd cryptography
```

## How to use
- For encrypting and decrypting files:
```cmd
cryptography [file1 file2 ... fileN] [encrypt|decrypt] [aes|des|rsa]
```

- For local encryption and decryption:
```cmd
cryptography
```
